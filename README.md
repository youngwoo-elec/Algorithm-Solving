# 🏆 Algorithm & Problem Solving
> **"Unshakable logic through rigorous verification and consistent practice"**

본 저장소는 다양한 알고리즘 문제를 해결하며 **문제 해결 전략(Problem Solving)**을 체계화하고, 
효율적인 알고리즘 설계를 통해 전자공학도에게 필요한 **최적화 사고 역량**을 기르는 공간입니다.

---

## 📌 Key Objectives
* **Algorithmic Efficiency:** 시간 및 공간 복잡도 분석을 통한 최적의 코드 작성
* **Data Structures:** Stack, Queue, Tree 등 자료구조의 실제 구현 및 적재적소 활용
* **Problem Decomposition:** 복잡한 문제를 작은 단위로 분해하고 논리적으로 검증하는 프로세스 정립
* **Continuous Growth:** 구름, 백준 등 다양한 플랫폼의 문제를 풀며 꾸준한 잔디 심기 실천

## 📂 Directory Structure
* `Goorm`: 구름 LEVEL 단계별 문제 풀이 기록
* `Baekjoon`: 백준 온라인 저지(BOJ) 난이도별 솔루션
* `Programmers`: 프로그래머스 코딩테스트 대비 문제
* `Algorithm-Concepts`: 정렬, 탐색, DP 등 주요 알고리즘 핵심 개념 정리
* `Notes`: 오답 분석 및 더 나은 접근 방식에 대한 기술적 회고

## 🛠 Tech Stack & Environment
* **Language:** C / C++
* **IDE:** Goorm IDE, GitHub Codespaces
* **Compiler:** GCC
